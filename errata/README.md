# Errata

If you find any mistakes in the first edition, *Unified SecOps Playbook, First Edition*, or if you have suggestions for improvements, then please [raise an issue in this repository](https://github.com/PacktPublishing/Unified-SecOps-Playbook/issues) or email the authors.

[**Errata**](errata.md): Typos, factual mistakes, interface changes, or mistakes in code.

All errata and improvements will be included in the next edition of the book.
