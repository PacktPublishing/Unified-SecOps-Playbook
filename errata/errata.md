# Chapter 3: Microsoft Security for Partners - Solutions Partner for Security

In mid-2025, [Microsoft Certified: Information Security Administrator Associate](https://learn.microsoft.com/credentials/certifications/information-security-administrator/) was added as a qualifying certification. This supercedes the [Microsoft Information Protection Administrator](https://learn.microsoft.com/en-us/certifications/information-protection-administrator/) certification, which retired on May 31st 2025. If the latter certification was passed before retirement, it remains valid for one year.

> Read more here: https://learn.microsoft.com/partner-center/membership/solutions-partner-security#intermediate-certifications-metric

# Chapter 3: Microsoft Security for Partners - Security advanced specializations

In 2022, Advanced Specializations were renamed to simply 'Specializations', dropping the 'Advanced' prefix.

> Read more here: https://blogs.microsoft.com/blog/2022/03/16/evolving-microsoft-partner-network-programs-for-partner-growth-and-customer-success/

# Chapter 3: Microsoft Security for Partners - PAL: get recognized for Azure impact

The new name for Microsoft Partner Network (MPN) ID is "partner network ID", often shortened to "partner ID".

More specifically, PAL uses the [partner location ID, as opposed to the partner global ID](https://learn.microsoft.com/partner-center/account-settings/account-structure).
